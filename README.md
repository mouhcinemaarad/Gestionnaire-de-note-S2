## 📝 Description
Une application desktop développée en **Python** utilisant la bibliothèque **Tkinter**. Ce projet permet de gérer et de calculer les notes du deuxième semestre (S2) pour les étudiants.

L'application automatise le calcul de la moyenne générale tout en vérifiant les conditions de validation du semestre, notamment la détection des notes éliminatoires.

## ✨ Fonctionnalités
* **Saisie intuitive** : Interface graphique simple pour entrer les notes de 7 modules.
* **Calcul Automatique** : Calcul instantané de la moyenne générale.
* **Logique de Validation** : Vérification automatique du statut du semestre (Validé / Non Validé).
* **Gestion d'erreurs** : Validation des entrées utilisateur.
